# O – Depth Progress

|  |  |
|---|---|
| **Goal** | Visualise descent from depth 1 → 50 |
| **Primary Data** | `character.stats.depth` |
| **UI** | Vertical progress bar + milestone markers |
| **Logic** | Animate depth change; tooltip shows % explored |
